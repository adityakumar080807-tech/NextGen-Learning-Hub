NextGen Learning Hub 🌐📚
A modern AI-powered educational website built using HTML, CSS, and JavaScript. This project was created to explore AI-assisted web development and demonstrate how AI tools can help generate responsive and professional websites efficiently.
🚀 Project Overview
NextGen Learning Hub is a responsive educational platform website designed with a modern UI/UX. The website includes multiple interactive sections such as courses, testimonials, statistics, features, and a contact form to simulate a real-world online learning platform.
This project was developed as part of an internship task focused on AI Website Generation.
✨ Features
Fully responsive design
Modern and clean UI
Sticky navigation bar
Smooth scrolling animations
Hero section with CTA buttons
About section with feature cards
Course showcase section
Interactive statistics counters
Testimonials section
Contact form with social links
Mobile-friendly hamburger menu
Hover effects and transitions
